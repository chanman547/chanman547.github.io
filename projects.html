CS 309:
For my CS 309 project I worked on creating a remote control Roomba that could autonomous map an environment.  There were 4 of us working on this project and I was in charge of developing the app that would control the Roomba.  For this project we used java for the app, C/C++ for the server, and also for the mapping algorithm.

CprE 381 Final Project:
For my CprE 381 final project we had to make a MIPS processor with a pipeline.  I worked with one other person for this project.  During this project we both worked on connecting all the components that we coded throughout the semester.  The final product of all of these components combined was the processor.  We did all of the coding for this project in VHDL.
